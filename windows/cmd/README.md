# cmd + clink

This folder captures the parts of the Windows cmd setup that are worth sharing:

- `command-processor.reg`: `AutoRun` injection for clink
- `console-settings.md`: console font, color, buffer, and window defaults
- `clink_settings`: clink behavior and color tuning
- `.inputrc`: readline behavior loaded by clink

What is intentionally excluded:

- history files
- logs
- generated runtime state
